# Server Administration & Version Control Guide

ဒီ Guide တွင် macOS ပေါ်မှာ SSH (Remote Access)၊ Apache Server နှင့် Git/GitHub တို့ကို စနစ်တကျ Setup လုပ်ပြီး အသုံးပြုပုံများကို အဆင့်ဆင့် ဖော်ပြပေးထားပါသည်။

---

## 1. SSH Server (Secure Shell)

အခြားစက်များမှ Remote access ရယူနိုင်ရန်အတွက် SSH ကို အသုံးပြုပါသည်။

### Enable/Disable SSH

* **SSH ဖွင့်ရန်:** `sudo systemsetup -setremotelogin on`
* **SSH ပိတ်ရန်:** `sudo systemsetup -setremotelogin off`

### Firewall Configuration (လုံခြုံရေးအတွက်)

macOS ၏ Firewall တွင် SSH port ကို ခွင့်ပြုပေးရန် လိုအပ်ပါသည်။

1. **Firewall ဖွင့်ရန်:** `sudo /usr/libexec/ApplicationFirewall/socketfilterfw --setglobalstate on`
2. **SSH ကို Allow လုပ်ရန်:** `sudo /usr/libexec/ApplicationFirewall/socketfilterfw --add /usr/sbin/sshd`
3. **Connection ကို Unblock လုပ်ရန်:** `sudo /usr/libexec/ApplicationFirewall/socketfilterfw --unblockapp /usr/sbin/sshd`

### Finding Your IP Address

Remote မှ လှမ်းချိတ်ရန် မိမိစက်၏ IP Address ကို အောက်ပါ command များဖြင့် ကြည့်နိုင်ပါသည်။

* **Wi-Fi IP:** `ipconfig getifaddr en0`
* **Ethernet IP:** `ipconfig getifaddr en1`

### SSH Client & Key Exchange

* **Login to Server:** `ssh [username]@[IPaddress]`
* **Generate SSH Key:** `ssh-keygen` (Password ရိုက်ရန်မလိုဘဲ Login ဝင်နိုင်ရန် Key ထုတ်ခြင်း)
* **Copy Key to Server:** `ssh-copy-id [username]@[IPaddress]`

:::tip Note
`ssh-copy-id` မရှိပါက `cat ~/.ssh/id_rsa.pub` ကိုသုံးပြီး Public Key ကို copy ယူကာ Server ၏ `authorized_keys` ဖိုင်ထဲသို့ manually ထည့်သွင်းနိုင်ပါသည်။
:::

## 2. Apache Web Server

Local Web Development ပြုလုပ်ရန် macOS တွင် built-in ပါဝင်သော Apache ကို အသုံးပြုပုံ ဖြစ်ပါသည်။

### Basic Commands

* **Check Version:** `httpd -v`
* **Start Server:** `sudo apachectl start`
* **Stop Server:** `sudo apachectl stop`
* **Restart Server:** `sudo apachectl restart`

### Configuration Paths

* **Default Web Root:** `/Library/WebServer/Documents/` (Website file များနှင့် `index.html` ထားရှိရာနေရာ)
* **Main Config File:** `sudo nano /etc/apache2/httpd.conf` (Server settings များ ပြင်ဆင်ရန်)

## 3. Git & GitHub

Version Control စနစ်ကို အသုံးပြုပြီး မိမိ၏ Code များကို စနစ်တကျ သိမ်းဆည်းနည်း ဖြစ်ပါသည်။

### Initial Configuration (ပထမဆုံးတစ်ကြိမ် ပြုလုပ်ရန်)

```bash
git config --global user.name "Htoo Aunk Linn"
git config --global user.email "your-email@example.com"
```

### Git Workflow Steps

1. **Check Version:** `git --version`
2. **Initialize Project:** `git init` (Folder တစ်ခုကို Git စတင်အသုံးပြုရန်)
3. **Stage Files:** `git add .` (ပြောင်းလဲမှုအားလုံးကို သိမ်းဆည်းရန် အသင့်ပြင်ခြင်း)
4. **Commit:** `git commit -m "1st commit"` (ပြောင်းလဲမှုများကို Version တစ်ခုအဖြစ် သတ်မှတ်ခြင်း)
5. **Connect to GitHub:** `git remote add origin https://github.com/your_username/repo_name.git`
6. **Branching:**
   * `git branch` (လက်ရှိ ရောက်ရှိနေသော branch ကို ကြည့်ရန်)
   * `git branch -a` (Branch အားလုံးကို ကြည့်ရန်)
