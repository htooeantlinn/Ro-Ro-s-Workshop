# Lesson 12: Practical Sensor Projects

Arduino Uno R3 သင်ခန်းစာတွေရဲ့ နောက်ဆုံးအပိုင်းမှာတော့ Sensor တွေကို အသုံးပြုပြီး လက်တွေ့ကျတဲ့ Project တွေကို ဘယ်လိုတည်ဆောက်မလဲဆိုတာ လေ့လာသွားမှာဖြစ်ပါတယ်။

### 1. Soil Moisture Sensor (မြေဆီလွှာ စိုထိုင်းဆ တိုင်းတာခြင်း)
အပင်တွေ ရေလောင်းဖို့ လိုမလို သိနိုင်ဖို့ မြေကြီးထဲမှာ ရေငွေ့ဘယ်လောက်ရှိသလဲဆိုတာကို LED အဆင့်ဆင့်နဲ့ ပြသပေးမည့် Project ဖြစ်ပါတယ်။



[Image of Arduino soil moisture sensor circuit diagram]


```cpp
int moisture = 0;

void setup() {
  pinMode(A0, OUTPUT); // Sensor ကို ပါဝါပေးရန်
  pinMode(A1, INPUT);  // Sensor Data ဖတ်ရန်
  Serial.begin(9600);
  
  // LED Pins များကို Output အဖြစ်သတ်မှတ်ခြင်း
  for (int pin = 8; pin <= 12; pin++) {
    pinMode(pin, OUTPUT);
  }
}

void loop() {
  // Sensor သံချေးတက်ခြင်းကို ကာကွယ်ရန် လိုအပ်မှသာ ပါဝါပေးခြင်း
  digitalWrite(A0, HIGH);
  delay(10); 
  moisture = analogRead(A1);
  digitalWrite(A0, LOW); // ဖတ်ပြီးလျှင် ပါဝါပြန်ပိတ်မည်

  Serial.println(moisture);

  // LED အားလုံးကို အရင်ပိတ်မည်
  for (int pin = 8; pin <= 12; pin++) digitalWrite(pin, LOW);

  // စိုထိုင်းဆအလိုက် LED တစ်လုံးချင်းစီ လင်းစေခြင်း
  if (moisture < 200) {
    digitalWrite(12, HIGH); // ခြောက်သွေ့လွန်းသည်
  } else if (moisture < 400) {
    digitalWrite(11, HIGH);
  } else if (moisture < 600) {
    digitalWrite(10, HIGH);
  } else if (moisture < 800) {
    digitalWrite(9, HIGH);
  } else {
    digitalWrite(8, HIGH);  // စိုစွတ်သည်
  }
  delay(100); 
}
```

### 2. Digital Voltage Meter (ဗို့မီတာ ပြုလုပ်ခြင်း)
7-Segment Display ကို အသုံးပြုပြီး ပတ်လမ်းတစ်ခုရဲ့ Voltage ကို တိုင်းတာပြသပေးမည့် Project ဖြစ်ပါတယ်။



```cpp
#include "Adafruit_LEDBackpack.h"

Adafruit_7segment led_display1 = Adafruit_7segment();

void setup() {
  led_display1.begin(0x70); // I2C address 112 = 0x70
  pinMode(A1, INPUT);
}

void loop() {
  // 0-1023 range ကို 0-30V range သို့ ပြောင်းလဲခြင်း
  // Formula: (analogRead * 30.0) / 1023.0
  float currentVoltage = analogRead(A1) * 0.0293255;

  led_display1.println(currentVoltage);
  led_display1.writeDisplay();
  delay(100); 
}
```

### 3. IR Receiver (အဝေးထိန်းခလုတ်ဖြင့် ထိန်းချုပ်ခြင်း)
TV Remote ကဲ့သို့သော Infrared Remote တွေကနေ ပေးပို့တဲ့ signal တွေကို ဖမ်းယူပြီး Display မှာ ပြသပေးမည့် နည်းလမ်းဖြစ်ပါတယ်။

```cpp
#include "Adafruit_LEDBackpack.h"
#include <IRremote.h>

Adafruit_7segment led_display1 = Adafruit_7segment();

void setup() {
  led_display1.begin(0x70);
  IrReceiver.begin(2); // IR Receiver ကို Pin 2 တွင် ချိတ်ထားသည်

  led_display1.println("----");
  led_display1.writeDisplay();
}

void loop() {
  // ခလုတ်နှိပ်လိုက်သည့် Signal ရောက်မရောက် စစ်ဆေးခြင်း
  if (IrReceiver.decode()) {
    unsigned long code = IrReceiver.decodedIRData.decodedRawData;
    
    // ရရှိလာသော Code ကို Display တွင် ပြသခြင်း
    led_display1.println(code, HEX);
    led_display1.writeDisplay();

    IrReceiver.resume(); // နောက်ထပ် signal ကို လက်ခံရန် ပြင်ဆင်ခြင်း
  }
  delay(10); 
}
```

### 💡 Pro Tips for Finishers
* **Metal Corrosion:** Soil moisture sensor တွေကို အမြဲတမ်း ပါဝါပေးမထားပါနဲ့ (Code ထဲကလို ခဏပဲပေးပါ)။ ဒါမှ Sensor ငုတ်တွေ သံချေးတက်တာ သက်သာမှာဖြစ်ပါတယ်။
* **I2C Communication:** 7-segment display နဲ့ LCD တွေ သုံးတဲ့အခါ `SDA` နဲ့ `SCL` ကြိုးတွေကို မမှားဖို့ သတိပြုပါ။
* **Libraries:** လက်တွေ့မှာ ဒီ Project တွေစမ်းသပ်တဲ့အခါ `IRremote` နဲ့ `Adafruit LED Backpack` library တွေကို Arduino IDE မှာ Install လုပ်ဖို့ မမေ့ပါနဲ့။
