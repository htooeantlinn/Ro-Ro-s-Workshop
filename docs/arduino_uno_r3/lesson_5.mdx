# Lesson 5: Advanced Analog Techniques

ဒီသင်ခန်းစာမှာတော့ Analog Input တွေကို ပိုမိုတိကျစေဖို့နဲ့ တခြား Device တွေကို ထိန်းချုပ်ဖို့အတွက် data တွေကို ဘယ်လို Processing လုပ်မလဲဆိုတာ လေ့လာသွားပါမယ်။

### 1. Analog In, Serial Out (Mapping Data)
Arduino ရဲ့ Analog Read (0-1023) ကို LED ရဲ့ Brightness (0-255) အဖြစ် ပြောင်းလဲအသုံးပြုနည်း ဖြစ်ပါတယ်။



```cpp
int sensorValue = 0;
int outputValue = 0;

void setup() {
  pinMode(A0, INPUT);
  pinMode(9, OUTPUT); // PWM Pin
  Serial.begin(9600);
}

void loop() {
  sensorValue = analogRead(A0);

  // map(value, fromLow, fromHigh, toLow, toHigh)
  // 0-1023 အကွာအဝေးကို 0-255 အဖြစ် အချိုးချပြောင်းလဲခြင်း
  outputValue = map(sensorValue, 0, 1023, 0, 255);

  analogWrite(9, outputValue);

  Serial.print("sensor = ");
  Serial.print(sensorValue);
  Serial.print("\t output = ");
  Serial.println(outputValue);
  
  delay(2); 
}
```

### 2. Sensor Calibration (အာရုံခံနိုင်စွမ်းကို ချိန်ညှိခြင်း)
Sensor တစ်ခုရဲ့ အနိမ့်ဆုံးနဲ့ အမြင့်ဆုံး တန်ဖိုးတွေကို ပတ်ဝန်းကျင် အခြေအနေအလိုက် ချိန်ညှိပေးတဲ့ နည်းလမ်းဖြစ်ပါတယ်။ စစချင်း ၅ စက္ကန့်အတွင်းမှာ Calibration လုပ်ပါမယ်။



```cpp
const int sensorPin = A0;
const int ledPin = 9;

int sensorValue = 0;
int sensorMin = 1023; // အနိမ့်ဆုံးကို အမြင့်ဆုံးတန်ဖိုးအရင်ထား
int sensorMax = 0;    // အမြင့်ဆုံးကို အနိမ့်ဆုံးတန်ဖိုးအရင်ထား

void setup() {
  pinMode(LED_BUILTIN, OUTPUT);
  digitalWrite(LED_BUILTIN, HIGH); // Calibration လုပ်နေကြောင်း မီးဖွင့်ပြမည်

  // ပထမ ၅ စက္ကန့်အတွင်း Sensor တန်ဖိုးများကို မှတ်သားခြင်း
  while (millis() < 5000) {
    sensorValue = analogRead(sensorPin);
    if (sensorValue > sensorMax) sensorMax = sensorValue;
    if (sensorValue < sensorMin) sensorMin = sensorValue;
    delay(50);
  }

  digitalWrite(LED_BUILTIN, LOW); // Calibration ပြီးကြောင်း မီးပိတ်ပြမည်
}

void loop() {
  sensorValue = analogRead(sensorPin);
  
  // Calibrated ဖြစ်ထားသော range အတိုင်း mapping လုပ်ခြင်း
  sensorValue = map(sensorValue, sensorMin, sensorMax, 0, 255);
  
  // တန်ဖိုးကို 0-255 အတွင်းသာ ရှိစေရန် ကန့်သတ်ခြင်း
  sensorValue = constrain(sensorValue, 0, 255);

  analogWrite(ledPin, sensorValue);
  delay(50);
}
```

### 3. Smoothing (Noise များဖယ်ရှားခြင်း)
Sensor ကလာတဲ့ data တွေဟာ တစ်ခါတလေ အရမ်းခုန်နေတတ်ပါတယ်။ အဲဒီအခါမှာ data ၁၀ ခုကို ပျမ်းမျှ (Average) ရှာပြီး ငြိမ်သွားအောင် လုပ်တဲ့ နည်းလမ်းဖြစ်ပါတယ်။



```cpp
const int numReadings = 10;
int readings[numReadings];      
int readIndex = 0;              
int total = 0;                  
int average = 0;                
int inputPin = A0;

void setup() {
  Serial.begin(9600);
  // Array ထဲက တန်ဖိုးများကို 0 အဖြစ် စတင်သတ်မှတ်ခြင်း
  for (int thisReading = 0; thisReading < numReadings; thisReading++) {
    readings[thisReading] = 0;
  }
}

void loop() {
  // အဟောင်းဆုံး data ကို နှုတ်ထုတ်မည်
  total = total - readings[readIndex];
  
  // data အသစ်ကို ဖတ်ပြီး array ထဲထည့်မည်
  readings[readIndex] = analogRead(inputPin);
  
  // data အသစ်ကို ပေါင်းထည့်မည်
  total = total + readings[readIndex];
  
  readIndex = readIndex + 1;

  // Array အဆုံးရောက်ရင် အစကနေ ပြန်သွားမည်
  if (readIndex >= numReadings) readIndex = 0;

  // ပျမ်းမျှတန်ဖိုး ရှာခြင်း
  average = total / numReadings;
  
  Serial.println(average);
  delay(1); 
}
```

:::info Summary
* **Map:** တန်ဖိုးတွေကို တစ်ခုကနေ တစ်ခု အချိုးချပြောင်းလဲချင်ရင် သုံးပါ။
* **Calibration:** Sensor တစ်ခုချင်းစီရဲ့ Range ကို တိကျချင်ရင် သုံးပါ။
* **Smoothing:** Serial Plotter မှာ ကြည့်တဲ့အခါ data တွေ အရမ်းခုန်နေရင် ငြိမ်သွားဖို့ သုံးပါ။
:::
