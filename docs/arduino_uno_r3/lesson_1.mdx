# Lesson 1: Basic Output Control

Arduino ရဲ့ အခြေခံအကျဆုံး Output control ဖြစ်တဲ့ Digital Write (အဖွင့်/အပိတ်) နဲ့ Analog Write (အလင်းအမှောင်ချိန်ခြင်း) သင်ခန်းစာ ဖြစ်ပါတယ်။

### 1. Breadboard Connection
အခြေခံ Circuit တစ်ခု တည်ဆောက်တဲ့အခါ လျှပ်စစ်စီးဆင်းမှုကို နားလည်ရန် အရေးကြီးပါတယ်။

```cpp
void setup() {
  pinMode(LED_BUILTIN, OUTPUT);
}

void loop() {
  digitalWrite(LED_BUILTIN, HIGH);
  delay(1000); 
  digitalWrite(LED_BUILTIN, LOW);
  delay(1000);
}
```

---

### 2. Blink (LED မှိတ်တုတ်မှိတ်တုတ်လုပ်ခြင်း)
Pin 13 မှာရှိတဲ့ Built-in LED ကို အသုံးပြုပြီး Code စတင်ရေးသားနည်း ဖြစ်ပါတယ်။

```cpp
void setup() {
  // Pin 13 ကို Output အဖြစ် သတ်မှတ်ခြင်း
  pinMode(LED_BUILTIN, OUTPUT);
}

void loop() {
  digitalWrite(LED_BUILTIN, HIGH); // LED မီးဖွင့်မည်
  delay(1000);                     // 1 စက္ကန့် စောင့်မည်
  digitalWrite(LED_BUILTIN, LOW);  // LED မီးပိတ်မည်
  delay(1000);                     // 1 စက္ကန့် စောင့်မည်
}
```

---

### 3. Fade (LED အလင်းအမှောင် အနှေးအမြန်ချိန်ခြင်း)
PWM (Pulse Width Modulation) ကို အသုံးပြုပြီး LED ရဲ့ brightness ကို ချိန်ညှိနည်း ဖြစ်ပါတယ်။ (PWM Pin ဖြစ်တဲ့ ~9 ကို သုံးထားပါတယ်)

```cpp
int brightness = 0;

void setup() {
  pinMode(9, OUTPUT);
}

void loop() {
  // LED ကို တဖြည်းဖြည်း လင်းလာစေရန်
  for (brightness = 0; brightness <= 255; brightness += 5) {
    analogWrite(9, brightness);
    delay(30); 
  }

  // LED ကို တဖြည်းဖြည်း မှိန်သွားစေရန်
  for (brightness = 255; brightness >= 0; brightness -= 5) {
    analogWrite(9, brightness);
    delay(30); 
  }
}
```

:::info Note
Uno R3 မှာ `~` သင်္ကေတပါတဲ့ Pin တွေ (3, 5, 6, 9, 10, 11) ဟာ PWM support ပေးတဲ့အတွက် Fade effect အတွက် ၎င်းတို့ကို သုံးရပါမယ်။
:::
