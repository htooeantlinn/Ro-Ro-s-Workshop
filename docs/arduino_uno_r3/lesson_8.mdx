# Lesson 8: Ultrasonic Distance Sensor

ဒီသင်ခန်းစာမှာတော့ Ultrasonic Sensor ကိုအသုံးပြုပြီး အရာဝတ္ထုတွေရဲ့ အကွာအဝေးကို စင်တီမီတာ (cm) သို့မဟုတ် လက်မ (inches) နဲ့ ဘယ်လိုတိုင်းတာမလဲဆိုတာ လေ့လာသွားပါမယ်။

### Ultrasonic Sensor ဘယ်လိုအလုပ်လုပ်သလဲ?
Sensor ရဲ့ Trigger pin ကနေ လူနားနဲ့မကြားနိုင်တဲ့ အသံလှိုင်း (Ultrasound) ကို လွှတ်လိုက်ပါတယ်။ အဲဒီအသံလှိုင်းဟာ အရာဝတ္ထုတစ်ခုခုကို ရိုက်မိပြီး ပြန်လာတဲ့အခါ Echo pin ကနေ ပြန်ဖမ်းယူပါတယ်။ အသံလှိုင်း အသွားအပြန် ကြာချိန်ကို ကြည့်ပြီး အကွာအဝေးကို တွက်ချက်တာ ဖြစ်ပါတယ်။



---

### Distance Measurement (အကွာအဝေးတိုင်းတာခြင်း)
ရှေ့မှာရှိတဲ့ အတားအဆီးနဲ့ အကွာအဝေးကို Serial Monitor မှာ ပြသပေးမည့် Code ဖြစ်ပါတယ်။

```cpp
int inches = 0;
int cm = 0;

// အကွာအဝေးကို တိုင်းတာပေးမည့် သီးသန့် function
long readUltrasonicDistance(int triggerPin, int echoPin) {
  pinMode(triggerPin, OUTPUT);  
  digitalWrite(triggerPin, LOW);
  delayMicroseconds(2);
  
  // အသံလှိုင်းစတင်လွှတ်ရန် Trigger ကို 10 microseconds ကြာ HIGH ပေးခြင်း
  digitalWrite(triggerPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(triggerPin, LOW);
  
  pinMode(echoPin, INPUT);
  // Echo pin မှ အသံလှိုင်းပြန်လာသည့် ကြာချိန်ကို microseconds ဖြင့် ဖတ်ခြင်း
  return pulseIn(echoPin, HIGH);
}

void setup() {
  Serial.begin(9600);
}

void loop() {
  // အကွာအဝေးကို cm ဖြင့် တွက်ချက်ခြင်း (0.01723 သည် အသံ၏အလျင်ကို မူတည်၍ တွက်ထားခြင်းဖြစ်သည်)
  // ဤနေရာတွင် Trigger နှင့် Echo ကို Pin 7 တစ်ခုတည်း သုံးထားသည် (3-pin sensor အမျိုးအစား)
  cm = 0.01723 * readUltrasonicDistance(7, 7);
  
  // cm မှ inches သို့ ပြောင်းလဲခြင်း
  inches = (cm / 2.54);

  Serial.print(inches);
  Serial.print(" in, ");
  Serial.print(cm);
  Serial.println(" cm");

  delay(100); 
}
```

### 💡 သိထားသင့်သည့် အချက်များ
* **Sensor အမျိုးအစား:** အသုံးအများဆုံး **HC-SR04** မှာ Pin ၄ ခု (VCC, GND, Trig, Echo) ပါဝင်ပြီး Tinkercad ရှိ အချို့ sensor များတွင် Pin ၃ ခု (VCC, GND, SIG) သာ ပါတတ်ပါတယ်။
* **Accuracy:** အသံလှိုင်းကို သုံးတာဖြစ်လို့ အသံလှိုင်းကို စုပ်ယူနိုင်တဲ့ အရာဝတ္ထုတွေ (ဥပမာ- ဂွမ်း၊ ရေမြှုပ်) ကို တိုင်းတာတဲ့အခါ တိကျမှု လျော့နည်းနိုင်ပါတယ်။
* **Range:** များသောအားဖြင့် 2cm ကနေ 400cm အထိ တိုင်းတာနိုင်ပါတယ်။
