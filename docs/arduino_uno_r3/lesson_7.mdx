# Lesson 7: Sound and Melodies

ဒီသင်ခန်းစာမှာတော့ Arduino ကို အသုံးပြုပြီး အသံလှိုင်း (Frequencies) တွေ ဘယ်လိုထုတ်မလဲဆိုတာ လေ့လာပါမယ်။ Speaker သို့မဟုတ် Buzzer တစ်ခုရှိရုံနဲ့ ကိုယ်ပိုင် Piano သေးသေးလေး ဒါမှမဟုတ် သီချင်းသံစဉ်လေးတွေ ဖန်တီးနိုင်ပါတယ်။

### 1. Tone Keyboard (Mini Piano)
ခလုတ် ၃ ခုကို အသုံးပြုပြီး မတူညီတဲ့ အသံ ၃ မျိုး (Note A, B, C) ထွက်အောင် လုပ်ဆောင်နည်း ဖြစ်ပါတယ်။



```cpp
void setup() {
  pinMode(A0, INPUT);
  pinMode(A1, INPUT);
  pinMode(A2, INPUT);
  pinMode(8, OUTPUT); // Speaker Pin
}

void loop() {
  // A0 ခလုတ်ကို နှိပ်လျှင် Note A4 (440Hz) အသံလွှတ်မည်
  if (digitalRead(A0) == HIGH) {
    tone(8, 440, 100); 
  }
  // A1 ခလုတ်ကို နှိပ်လျှင် Note B4 (494Hz) အသံလွှတ်မည်
  if (digitalRead(A1) == HIGH) {
    tone(8, 494, 100); 
  }
  // A2 ခလုတ်ကို နှိပ်လျှင် Note C5 (523Hz) အသံလွှတ်မည်
  if (digitalRead(A2) == HIGH) {
    tone(8, 523, 100); 
  }
  delay(10); 
}
```

### 2. Tone Melody (သီချင်းသံစဉ်တီးခတ်ခြင်း)
သတ်မှတ်ထားတဲ့ Note တွေကို Array ထဲထည့်ပြီး သီချင်းတစ်ပုဒ်အဖြစ် အလိုအလျောက် တီးခတ်ခိုင်းနည်း ဖြစ်ပါတယ်။

```cpp
#define NOTE_C4  262
#define NOTE_G3  196
#define NOTE_A3  220
#define NOTE_B3  247

int melody[] = { NOTE_C4, NOTE_G3, NOTE_G3, NOTE_A3, NOTE_G3, 0, NOTE_B3, NOTE_C4 };
int noteDurations[] = { 4, 8, 8, 4, 4, 4, 4, 4 };

void setup() {
  for (int thisNote = 0; thisNote < 8; thisNote++) {
    // အသံကြာချိန်ကို တွက်ချက်ခြင်း
    int noteDuration = 1000 / noteDurations[thisNote];
    tone(8, melody[thisNote], noteDuration);

    // အသံတစ်ခုနှင့်တစ်ခုကြား ခေတ္တရပ်ခြင်း
    int pauseBetweenNotes = noteDuration * 1.30;
    delay(pauseBetweenNotes);
    noTone(8); // အသံရပ်မည်
  }
}

void loop() {
  // setup ထဲမှာ ရေးထားတဲ့အတွက် တစ်ကြိမ်သာ တီးပါမည်
}
```

### 3. Multiple Tone Player
Speaker အများအပြားကို တစ်ခုပြီးတစ်ခု အလှည့်ကျ အသံမြည်အောင် စေခိုင်းနည်း ဖြစ်ပါတယ်။

```cpp
void setup() {
  pinMode(6, OUTPUT);
  pinMode(7, OUTPUT);
  pinMode(8, OUTPUT);
}

void loop() {
  noTone(8);
  tone(6, 880, 200); // Pin 6 မှ အသံမြည်မည်
  delay(200); 

  noTone(6);
  tone(7, 988, 500); // Pin 7 မှ အသံမြည်မည်
  delay(500); 

  noTone(7);
  tone(8, 1047, 300); // Pin 8 မှ အသံမြည်မည်
  delay(300); 
}
```

### 4. Tone Pitch Follower (Light to Sound)
Sensor (ဥပမာ- LDR) ကနေ ဖတ်ရတဲ့ အလင်းပမာဏအပေါ် မူတည်ပြီး အသံရဲ့ အနိမ့်အမြင့် (Pitch) ကို ပြောင်းလဲပေးတာ ဖြစ်ပါတယ်။ အလင်းများရင် အသံစူးပြီး၊ အလင်းနည်းရင် အသံအစ်သွားပါလိမ့်မယ်။



```cpp
int sensorReading = 0;

void setup() {
  pinMode(A0, INPUT);
  Serial.begin(9600);
  pinMode(9, OUTPUT);
}

void loop() {
  sensorReading = analogRead(A0);
  Serial.println(sensorReading);

  // Sensor reading ကို မူတည်ပြီး Frequency အဖြစ် ပြောင်းလဲခြင်း
  int pitch = map(sensorReading, 0, 1023, 100, 1000);
  tone(9, pitch, 10); 
  
  delay(10); 
}
```

:::info Summary
* **tone(pin, frequency, duration):** အသံလွှတ်ရန် သုံးသည်။
* **noTone(pin):** အသံရပ်ရန် သုံးသည်။
* **Frequency:** ကိန်းဂဏန်းကြီးလေ အသံစူး (High pitch) လေ ဖြစ်သည်။
:::
