# Lesson 2: Input Handling (Buttons)

ဒီသင်ခန်းစာမှာတော့ ပတ်ဝန်းကျင်က အချက်အလက်တွေကို Arduino က ဘယ်လို လက်ခံရယူသလဲဆိုတဲ့ Digital Input အကြောင်းကို လေ့လာသွားပါမယ်။

### 1. Basic Button Control
Button ကို ဖိထားတဲ့အချိန်မှာ မီးလင်းစေပြီး၊ ပြန်လွှတ်လိုက်ရင် မီးပိတ်သွားစေတဲ့ အခြေခံ Code ဖြစ်ပါတယ်။



```cpp
int buttonState = 0;

void setup() {
  pinMode(2, INPUT);
  pinMode(LED_BUILTIN, OUTPUT);
}

void loop() {
  // Pin 2 က အချက်အလက်ကို ဖတ်မည်
  buttonState = digitalRead(2);

  // Button ဖိထားလျှင် (HIGH ဖြစ်လျှင်) မီးဖွင့်မည်
  if (buttonState == HIGH) {
    digitalWrite(LED_BUILTIN, HIGH);
  } else {
    digitalWrite(LED_BUILTIN, LOW);
  }
  delay(10); 
}
```

### 2. Debouncing (Signal Noise များကို ဖြေရှင်းခြင်း)
ခလုတ်တစ်ခုကို နှိပ်လိုက်တဲ့အခါ စက်ပိုင်းဆိုင်ရာအရ Signal တွေက တုန်ခါမှု (Noise) ဖြစ်တတ်ပါတယ်။ အဲဒီ Noise တွေကို ကျော်ဖြတ်ပြီး ခလုတ်နှိပ်တာ တိကျစေဖို့ Debounce လုပ်နည်းကို သုံးရပါတယ်။

```cpp
const int buttonPin = 2;    
const int ledPin = 13;      

int ledState = HIGH;         
int buttonState;             
int lastButtonState = LOW;   

unsigned long lastDebounceTime = 0;  
unsigned long debounceDelay = 50;    

void setup() {
  pinMode(buttonPin, INPUT);
  pinMode(ledPin, OUTPUT);
  digitalWrite(ledPin, ledState);
}

void loop() {
  int reading = digitalRead(buttonPin);

  // Signal ပြောင်းလဲမှုရှိမရှိ စစ်ဆေးခြင်း
  if (reading != lastButtonState) {
    lastDebounceTime = millis();
  }

  // သတ်မှတ်ထားတဲ့ Delay ထက်ကျော်မှ ခလုတ်အစစ်အမှန်ဟု သတ်မှတ်ခြင်း
  if ((millis() - lastDebounceTime) > debounceDelay) {
    if (reading != buttonState) {
      buttonState = reading;

      if (buttonState == HIGH) {
        ledState = !ledState; // မီးအခြေအနေကို ပြောင်းပြန်လှန်မည်
      }
    }
  }

  digitalWrite(ledPin, ledState);
  lastButtonState = reading;
}
```

### 3. State Change Detection (နှိပ်လိုက်တဲ့အကြိမ်ရေကို ရေတွက်ခြင်း)
ခလုတ်ကို ဖိထားတာကို မကြည့်ဘဲ "အဖွင့်ကနေ အပိတ်" သို့မဟုတ် "အပိတ်ကနေ အဖွင့်" ပြောင်းလဲသွားတဲ့ အခိုက်အတန့် (Edge Detection) ကို ဖမ်းယူတဲ့ နည်းလမ်းဖြစ်ပါတယ်။



```cpp
int buttonState = 0;
int lastButtonState = 0;
int buttonPushCounter = 0;

void setup() {
  pinMode(2, INPUT);
  Serial.begin(9600);
  pinMode(LED_BUILTIN, OUTPUT);
}

void loop() {
  buttonState = digitalRead(2);

  // အရင်အခြေအနေနဲ့ မတူတော့လျှင် (ပြောင်းလဲမှုရှိလျှင်)
  if (buttonState != lastButtonState) {
    if (buttonState == HIGH) {
      // ခလုတ်နှိပ်လိုက်သည့် အကြိမ်ရေကို တိုးမည်
      buttonPushCounter += 1;
      Serial.print("Number of button pushes: ");
      Serial.println(buttonPushCounter);
    }
    delay(5); 
  }
  
  lastButtonState = buttonState;

  // ၄ ကြိမ်နှိပ်တိုင်း တစ်ခါ မီးလင်းစေရန် (Modulo % သုံးထားသည်)
  if (buttonPushCounter % 4 == 0) {
    digitalWrite(LED_BUILTIN, HIGH);
  } else {
    digitalWrite(LED_BUILTIN, LOW);
  }
}
```

:::tip Pro Tip
ခလုတ်တစ်ခုကို နှိပ်လိုက်တိုင်း ကိန်းဂဏန်းတွေ တိုးသွားချင်တာမျိုး (ဥပမာ- Digital Counter) လုပ်ချင်ရင် **State Change Detection** ကို သုံးရပါတယ်။
:::
