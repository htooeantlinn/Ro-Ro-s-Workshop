# Lesson 11: Advanced Logic & Optimization

ဒီသင်ခန်းစာမှာတော့ Arduino ရဲ့ ဖတ်ရှုရရှိတဲ့ data တွေကို Voltage အဖြစ် ပြောင်းလဲတွက်ချက်နည်းနဲ့ Code တွေကို ပိုမိုမြန်ဆန် စမတ်ကျလာစေမယ့် နည်းလမ်းတွေကို လေ့လာပါမယ်။

### 1. Read Analog Voltage (Voltage တိုက်ရိုက်တွက်ချက်ခြင်း)
Analog Read ကနေရတဲ့ 0-1023 တန်ဖိုးကို တကယ့်လက်တွေ့ Voltage (0V - 5V) အဖြစ် သင်္ချာနည်းဖြင့် ပြောင်းလဲနည်း ဖြစ်ပါတယ်။



```cpp
void setup() {
  Serial.begin(9600);
}

void loop() {
  int sensorValue = analogRead(A0);

  // 0-1023 တန်ဖိုးကို 0-5V အဖြစ် ပြောင်းလဲတွက်ချက်ခြင်း
  // Formula: (Value * 5.0) / 1023.0
  float voltage = sensorValue * (5.0 / 1023.0);

  Serial.print("Voltage: ");
  Serial.println(voltage);
  delay(100);
}
```

### 2. Blink Without Delay (Multitasking လုပ်နည်း)
`delay()` ကို သုံးရင် Arduino ဟာ အဲဒီအချိန်အတွင်းမှာ တခြားအလုပ် ဘာမှမလုပ်ဘဲ ရပ်နေတတ်ပါတယ်။ အဲဒီပြဿနာကို ဖြေရှင်းဖို့ `millis()` (အချိန်မှတ်နာရီ) ကို အသုံးပြုပြီး အလုပ်နှစ်ခုကို တစ်ပြိုင်နက် လုပ်ဆောင်နည်း ဖြစ်ပါတယ်။



```cpp
const int ledPin = LED_BUILTIN;
int ledState = LOW;
unsigned long previousMillis = 0; // နောက်ဆုံး မီးမှိတ်ခဲ့သည့် အချိန်ကို မှတ်ထားရန်
const long interval = 1000;       // မီးမှိတ်တုတ်လုပ်မည့် ကြာချိန်

void setup() {
  pinMode(ledPin, OUTPUT);
}

void loop() {
  unsigned long currentMillis = millis(); // လက်ရှိအချိန်ကို ဖတ်ခြင်း

  // သတ်မှတ်ထားတဲ့ interval ပြည့်ပြီလား စစ်ဆေးခြင်း
  if (currentMillis - previousMillis >= interval) {
    previousMillis = currentMillis; // အချိန်ဟောင်းကို Update လုပ်မည်

    // LED အခြေအနေကို ပြောင်းပြန်လှန်မည်
    ledState = (ledState == LOW) ? HIGH : LOW;
    digitalWrite(ledPin, ledState);
  }
  
  // ဤနေရာတွင် delay() မရှိသည့်အတွက် တခြား code များကို တစ်ပြိုင်နက် ရေးသားနိုင်သည်
}
```

### 3. Input Pullup (Circuit ကို ရိုးရှင်းစေခြင်း)
အရင်သင်ခန်းစာတွေမှာ ခလုတ် (Button) သုံးရင် Resistor တစ်လုံး အပြင်မှာ ခံရပါတယ်။ `INPUT_PULLUP` ကို သုံးရင်တော့ Arduino ထဲမှာပါတဲ့ Internal Resistor ကို သုံးမှာဖြစ်လို့ အပြင်မှာ Resistor ခံစရာ မလိုတော့ပါဘူး။



```cpp
void setup() {
  Serial.begin(9600);
  
  // ခလုတ်ကို Resistor မလိုဘဲ တိုက်ရိုက်ချိတ်ရန် (Internal Pull-up သုံးမည်)
  pinMode(2, INPUT_PULLUP);
  pinMode(LED_BUILTIN, OUTPUT);
}

void loop() {
  int sensorVal = digitalRead(2);
  Serial.println(sensorVal);

  // သတိပြုရန်- INPUT_PULLUP သည် Logic ပြောင်းပြန်ဖြစ်သည်
  // ခလုတ်မနှိပ်လျှင် HIGH ဖြစ်ပြီး၊ နှိပ်လိုက်မှ LOW ဖြစ်သွားမည်
  if (sensorVal == LOW) { // ခလုတ်နှိပ်လိုက်လျှင်
    digitalWrite(LED_BUILTIN, HIGH);
  } else {
    digitalWrite(LED_BUILTIN, LOW);
  }
}
```

### 💡 သိထားသင့်သည့် အချက်များ
* **Floating Pin:** ခလုတ်တစ်ခုကို Resistor မခံဘဲ `INPUT` ရိုးရိုးပဲ သုံးရင် Pin က Signal တွေ ငြိမ်မနေဘဲ (0 နဲ့ 1 ကြား ခုန်နေပြီး) Error တက်တတ်ပါတယ်။ ဒါကြောင့် `INPUT_PULLUP` ကို သုံးတာ ဖြစ်ပါတယ်။
* **millis() vs delay():** Project အကြီးတွေမှာ `delay()` ကို ရှောင်ပြီး `millis()` ကို သုံးတာဟာ ပိုပြီး Professional ကျတဲ့ နည်းလမ်းဖြစ်ပါတယ်။

:::tip Summary
* **Voltage calculation:** Sensor ကနေ တိကျတဲ့ output voltage သိချင်ရင် သုံးပါ။
* **Millis:** အလုပ်တွေကို တစ်ပြိုင်နက် (Parallel) လုပ်ချင်ရင် သုံးပါ။
* **Input Pullup:** ခလုတ်ချိတ်တဲ့အခါ ကြိုးရှုပ်တာနဲ့ Resistor သုံးရတာ သက်သာချင်ရင် သုံးပါ။
:::
