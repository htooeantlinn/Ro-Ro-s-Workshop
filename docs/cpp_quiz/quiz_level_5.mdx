## Level 5:

### (1)
```cpp
int a = 6;
switch(a){
    case 1 ... 5: cout << "One to Five\n"; break;
    case 6 ... 8: cout << "Six to Eight\n"; break;
    case 9: cout << "Nine\n"; break;
    case 10: cout << "Ten\n"; break;
}
```
1. `One to Five`
2. `Six to Eight`
3. `Error`

<details>
  <summary>Click to see Answer</summary>
  <div>
    **Correct Answer: 2. Six to Eight**
    > `a` is 6, which falls within the range of the second case.
  </div>
</details>

### (2)
```cpp
float b = 5;
switch(b){
    case 3: cout << "Three!"; break;
    case 5: cout << "Five!";
    case 7: cout << "Seven!";
    default: cout << "Bye Bye!";
}
```
1. `Five!`
2. `Five!Seven!Bye Bye!`
3. `Error`

<details>
  <summary>Click to see Answer</summary>
  <div>
    **Correct Answer: 3. Error**
    > C++ `switch` statements only work with integral types (int, char, enum). `float` is not allowed.
  </div>
</details>

---

### (3)
```cpp
int n = 0;
switch(n){
    default: n++;
    case 1: ++n; break;
}
cout << n;
```
1. `0`
2. `1`
3. `2`
4. `Error`

<details>
  <summary>Click to see Answer</summary>
  <div>
    **Correct Answer: 3. 2**
    > No case matches 0, so it hits `default`. `n` becomes 1, then falls through to `case 1` where `n` becomes 2.
  </div>
</details>
