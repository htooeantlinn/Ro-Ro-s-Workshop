### (4)
```cpp
int a = 5;
cout << a;
int b = 7;
a++; b++;
cout << a << b;
```
1. `568`
2. `560`
3. `550`
4. `Error`

<details>
  <summary>Click to see Answer</summary>
  <div>
    **Correct Answer: 1. 568**
    > Prints 5, then increments. a becomes 6, b becomes 8. Final output: 568.
  </div>
</details>

### (5)
```cpp
int a = 5;
cout << a;
{
    int b = 7;
    a++; b++;
}
cout << a << b;
```
1. `568`
2. `560`
3. `550`
4. `Error`

<details>
  <summary>Click to see Answer</summary>
  <div>
    **Correct Answer: 4. Error**
    > `b` is declared inside a nested scope `{ }`, making it inaccessible to the final `cout`.
  </div>
</details>

### (6)
```cpp
int a = 1;
int b = ++a;
cout << a << b;
a++; ++b;
cout << a << b;
```
1. `1233`
2. `2233`
3. `2223`
4. `Compilation Error`

<details>
  <summary>Click to see Answer</summary>
  <div>
    **Correct Answer: 2. 2233**
    > First, `a` and `b` both become 2. Then they are incremented again to become 3.
  </div>
</details>
