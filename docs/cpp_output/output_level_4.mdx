### (3)
```cpp
int x = 4, y = 5;
if (++x > 10 && --y > 10)
    cout << y;
else
    cout << x;
```

<details>
  <summary>Click to see Answer</summary>
  <div>
```sh
5
```
    > Short-circuit: `++x` (5) is not > 10, so `--y` never runs. Prints `x`.
  </div>
</details>

### (4)
```cpp
int a = 2, b = 2;

    cout << a << b;
```

<details>
  <summary>Click to see Answer</summary>
  <div>
```sh
32
```
    > Short-circuit: `++a` is 3. 3==3 is true, so `||` stops there. `b` remains 2.
  </div>
</details>
